# 普惠金融进万家金融科普平台V1.0

## 项目简介

"普惠金融进万家"是由贵州大学金融专业学生发起的公益科普项目，依托"财经研习社"小红书账号（13.9万粉丝），打造"专业内容生产+新媒体轻量化传播+线上线下联动"的普惠金融科普体系。

## 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **状态管理**: Zustand
- **样式方案**: Tailwind CSS
- **图标库**: Lucide React

## 功能特性

### 1. 首页
- 顶部轮播图展示项目亮点
- 核心数据看板（科普人次、测评人数、活动场次）
- 快捷功能入口
- 热门科普文章推荐
- 底部横幅

### 2. 防骗知识库
- 分类标签导航（金融防骗、理财认知、投资基础、宏观金融）
- 文章列表展示
- 文章详情页（含点赞、分享功能）
- 相关文章推荐

### 3. 在线测评
- 测评规则说明
- 20道单选题互动答题
- 实时进度展示
- 测评结果报告（分数、等级、建议）
- 证书下载、分享功能

### 4. 关于我们
- 项目介绍
- 核心团队展示
- 指导老师介绍
- 合作单位
- 联系信息

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 项目结构

```
src/
├── components/          # 可复用组件
│   ├── BottomNav.tsx   # 底部导航栏
│   ├── Carousel.tsx    # 轮播图组件
│   ├── DataCard.tsx    # 数据卡片组件
│   ├── QuickEntry.tsx  # 快捷入口组件
│   ├── ArticleCard.tsx # 文章卡片组件
│   ├── CategoryTabs.tsx# 分类标签组件
│   └── TeamCard.tsx    # 团队卡片组件
├── pages/              # 页面组件
│   ├── Home.tsx        # 首页
│   ├── Knowledge.tsx   # 防骗知识库
│   ├── ArticleDetail.tsx# 文章详情
│   ├── Quiz.tsx        # 在线测评
│   └── About.tsx       # 关于我们
├── store/              # 状态管理
│   └── useStore.ts     # Zustand store
├── data/               # 模拟数据
│   ├── articles.ts     # 文章数据
│   ├── questions.ts    # 测评题目数据
│   └── team.ts         # 团队数据
├── App.tsx             # 应用主组件
├── main.tsx            # 应用入口
└── index.css           # 全局样式
```

## 设计规范

### 配色方案
- **主色**: #165DFF（腾讯蓝，金融专业感）
- **辅助色**: #FF7D00（橙色，公益温暖感）
- **文字色**: #1D2129（正文）、#4E5969（次要文字）
- **背景色**: #F2F3F5（页面背景）、#FFFFFF（卡片背景）

### 响应式设计
- 移动端优先设计
- 适配主流移动设备尺寸
- 触摸友好的交互体验
